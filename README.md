# Introduce
A tool to search information of stock !
# Install
```powershell
go get -u github.com/ainilili/stock
```
# Usage
```powershell
[root@nico ~]# stock list ziguangguowei
Name            Code
紫光国微        sz002049

[root@nico ~]# stock get sz002049
Name            Code            Price           Change
紫光国微        sz002049        115.000         1.77%
```
